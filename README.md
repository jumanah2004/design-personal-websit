# 🌐 My Personal Website

A simple, elegant personal portfolio website built with HTML and CSS to showcase my profile, skills, and projects.

---

## 🔗 Live Demo

✨ [Visit my Live Website here!](https://jumanah.free.je/)

---

## ⚙️ How It Works

The website is structured into several clean, semantic sections:

* Header & Profile: Highlights my profile picture, name, and specialization.
* Navigation Bar: Provides smooth navigation between different sections of the page.
* Skills Section: Highlights technical skills and areas of expertise using styled <span> elements for a modern badge look.
* Responsive Images & Layout: Utilizes modern CSS principles (like `max-width`) to ensure the site looks great on all screen sizes.

---

## 🛠️ Technologies Used

* HTML5 - Markup and page structure.
* CSS3 - Styling, layout, and responsiveness.

---

## 🚀 How to Run Locally

1. Download or Clone the project files.
2. Open the index.html file directly in any web browser.
